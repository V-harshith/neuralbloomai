# Product Overview

This is a business/agency website built with v0.app - an AI-powered web development platform. The site features a modern landing page with sections for services, team, portfolio, testimonials, and contact information.

## Key Features

- Responsive design with mobile navigation
- Smooth scrolling between sections
- Dark/light theme support
- Analytics integration via Vercel
- Professional business presentation layout

## Deployment

The project is automatically synced with v0.app deployments and hosted on Vercel. Changes made in v0.app are automatically pushed to the repository.
